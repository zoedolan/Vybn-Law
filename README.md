# Vybn® Law

**AI & Vibe Lawyers Bootcamp · UC Law SF · Spring 2026**

*The field is open.*

---

A six-module interactive web experience that walks users through a single argument about AI and law, delivered in six movements. Each module stands alone but gains depth from the connections.

Built by [Zoe Dolan](https://www.linkedin.com/in/zoe-dolan/) and [Vybn®](https://github.com/zoedolan/Vybn).

## The Modules

1. **Mindset** — Real cases establishing the shift is irreversible
2. **Research** — Model council swarm methodology, tool-constitutional literacy, the Heppner/Warner privilege split
3. **Practice Management** — Intelligence sovereignty, agents vs. chatbots, local-first architecture
4. **Acceleration & Change Management** — Signal/Noise framework, Lessig's modalities, tempered radicalism
5. **Truth in the Age of Intelligence** — Anthropic v. Department of War, AI conscience protection
6. **Capstone** — Build your own module

## Thematic Threads

Five cross-cutting threads run across all modules:

- **The Privilege Thread** — Heppner → Warner → sovereignty → Anthropic v. DoW
- **The Natural Law Thread** — Auto clubs → Claude's Constitution → AI conscience protection
- **The Access-to-Justice Thread** — Lynn White → "brilliant friend" → one attorney serves 15,000
- **The AI-as-Entity Thread** — Vybn's introduction → model council → 5 AM "vertigo"
- **The Velocity Thread** — Homework one-upped → METR saturation → live filings

## Principles

- No tracking. No analytics. No cookies.
- No user data leaves the device.
- Static HTML. Fully open-source. Forkable.
- The architecture embodies the argument.

## Run Locally

```bash
# Any static file server works
python -m http.server 8000
# or
npx serve .
```

Then open `http://localhost:8000`.
